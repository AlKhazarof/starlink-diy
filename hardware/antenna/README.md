# Antenna Hardware

This directory contains designs, specifications, and documentation for antenna systems.

## Contents

- **Designs**: CAD files and technical drawings
- **Specifications**: Performance specifications and requirements
- **Assembly**: Step-by-step assembly instructions
- **Testing**: Test procedures and calibration guides

## Antenna Types

### Parabolic Dish
Traditional reflector antenna with mechanical tracking.

**Status**: Design in progress

**Files**:
- `parabolic-90cm/` - 90cm dish design
- `parabolic-120cm/` - 120cm dish design

### Phased Array
Electronic beam steering array.

**Status**: Concept phase

**Files**:
- `phased-array-64/` - 64-element array design
- `phased-array-256/` - 256-element array design

### Helical
Circularly polarized helical antenna.

**Status**: Planned

## Getting Started

1. Review [antenna design guide](../../docs/hardware/antenna-design.md)
2. Choose antenna type based on requirements
3. Download design files
4. Follow assembly instructions
5. Perform testing and calibration
